界面初始化后自动加载 data-provide="datetimepicker-inline" 的标签
Usage 1:

<div data-provide="datetimepicker-inline" class="input-group date" data-date-autoclose="true" data-date-today-btn="true"
	data-start-view="2" data-min-view="2" data-date-format="yyyy-mm-dd">
	<input size="16" type="text" value="" readonly class="form-control">
	<span class="input-group-addon"><span class="glyphicon glyphicon-th glyphicon-remove"></span></span>
</div>
	
<input data-provide="datetimepicker-inline" type="text" value="" class="form-control" data-date-format="yyyy-mm" 
	data-start-view="3" data-min-view="3">
	
<input data-provide="datetimepicker-inline" type="text" value="" class="form-control" data-date-format="yyyy" 
	data-start-view="4" data-min-view="4">
Usage 2:

<input id="datepicker1" type="text" value="" class="form-control" readonly="readonly"/>

$('#datepicker1').datetimepicker({
	startView: 2,
	minView: 2,
	autoclose: true, //选择后是否自动关闭
	todayBtn: true, //显示当天
	format: 'yyyy-mm-dd' //返回的格式
}).on('changeDate', function(ev) {
	//dt是日期的工具类，详细请查看siov-date.js
	alert(dt.getDateStr(ev.date, 'yyyy-mm-dd'));
})
Property:
Name Html Name Type Description Default Usage
format data-date-format String 返回的日期格式 mm/dd/yyyy {format:'yyyy-mm-dd'} or data-date-format="yyyy-mm-dd"
startView data-start-view Number,String 日期时间选择器打开之后首先显示的视图(0:小时、1:日期、2:月、3:年：4：10年) 2(月) {startView:2} or data-start-view="2"
minView data-min-view Number,String 日期选择向下精准到什么程度，如默认0（属性值与startView相同含义），则一直选择到小时为止 0(小时) {minView:2} or data-min-view="2"
weekStart data-date-weekstart String 日历中显示从哪一天开始。0（星期日）到6（星期六） 0(星期日) {weekStart:1} or data-date-weekstart="1"
startDate data-date-startdate Date,String 日历中显示的最早的时间，小于此时间则禁用 空 {startDate:new Date()} or data-date-startdate="2016-11-15"
endDate data-date-enddate Date,String 日历中显示的最晚的时间，大于此时间则禁用 空 {endDate:new Date()} or data-date-enddate="2016-11-15"
daysOfWeekDisabled data-date-days-of-week-disabled Array 每周的哪一天禁用 空 {daysOfWeekDisabled:[0,6]} or data-date-days-of-week-disabled="0,6"
autoclose data-date-autoclose Boolean 选中日期后是否关闭选择框 false {autoclose:true} or data-date-autoclose="true"
todayBtn data-date-today-btn Boolean 是否在日历底部显示“今日”按钮 false {todayBtn:true} or data-date-today-btn="true"
todayHighlight(实测不生效) data-date-today-highligh Boolean 是否高亮显示当前日期 false {todayHighlight:true} or data-date-today-highligh="true"
keyboardNavigation data-date-keyboard-navigation Boolean 是否允许通过方向键改变日期 true {keyboardNavigation:true} or data-date-keyboard-navigation="true"
Method:
Name Parameter Description Return Usage
remove 无 销毁对象 无 $('#datetimepicker').datetimepicker('remove');
show 无 显示日期时间选择器 无 $('#datetimepicker').datetimepicker('show');
hide 无 隐藏日期时间选择器 无 $('#datetimepicker').datetimepicker('hide');
update 无 使用当前输入框中的值更新日期时间选择器 无 $('#datetimepicker').datetimepicker('update');
setStartDate startDate (String) 给日期时间选择器设置一个新的起始日期 无 $('#datetimepicker').datetimepicker('setStartDate', '2012-01-01'); $('#datetimepicker').datetimepicker('setEndDate');//取消
setEndDate endDate (String) 给日期时间选择器设置结束日期 无 $('#datetimepicker').datetimepicker('setEndDate', '2012-01-01'); $('#datetimepicker').datetimepicker('setEndDate');//取消
setDaysOfWeekDisabled daysOfWeekDisabled (String|Array) 设置禁选的日期（每周几） 无 $('#datetimepicker').datetimepicker('setDaysOfWeekDisabled', [0,6]); or $('#datetimepicker').datetimepicker('setDaysOfWeekDisabled'); //取消
Event:
Name Description Return Usage
changeDate 当日期被改变时被触发 event{data:Date} $('#date-end').datetimepicker().on('changeDate', function(ev){ alert(ev.date); });
