siov-select 下拉选择框控件
界面效果

多选：
引入模块
bs.loadScript('../', ['select'])
界面初始化后自动加载 data-toggle="select" 的div标签
Usage 1:

<div data-toggle="select" class="siov-select" data-show-null-item="true" data-empty-text="全选" 
  data-url="http://127.0.0.1:3000/base/getCountry" style="width:150px"></div>
Usage 2:

<div id="select1" class="siov-select" style="width:150px"></div>

$('#select1').initSelect({
	url: 'http://127.0.0.1:3000/base/getCountry',
	showNullItem: true,
	emptyText: '全选',
	onChange: function(id, text) {
	}
});
Property:
Name Html Name Type Description Default
data 无（需在JS中定义） Array 本地数据源,与url冲突 []
url data-url String 服务数据源，与data冲突 空
idField data-id-field String 下拉框唯一字段 id
textField data-text-field String 下拉框文本字段 text
showNullItem data-show-null-item Boolean 是否显示空选项 false
emptyText data-empty-text String 空选项显示的文本，showNullItem设置为true此属性生效 空
multiSelect data-multi-select Boolean 是否设置为多选 false
onChange 无（需在JS中定义） Function 值改变时的回调,返回选中的id，text onChange: function(id, text) {}
Test Method(请先初始化控件)
Method:
Name Parameter Description Return Usage Test
initSelect Options(参见属性列表) 初始化控件，若控件已初始化过，此方法失效 无 $('#select1').initSelect()
getSelectValue 无 获取控件已选择的值，若未选择返回空 selectedValue(String) $('#select1').getSelectValue()
getSelectText 无 获取控件已选择的文本，若未选择返回空 selectedText(String) $('#select1').getSelectText()
select selectValue(String) 指定控件选择某个值 无 $('#select1').select('zh')
getSelectAllData 无 获取当前下拉框所有数据 Array $('#select1').getSelectAllData()
reloadSelect Options(参见属性列表) 重新初始化select控件 空 $('#select1').reloadSelect({})
destorySelect 无 销毁控件 无 $('#select1').destorySelect()
