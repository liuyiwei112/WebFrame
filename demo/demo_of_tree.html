bootstrap-tree 树控件
界面效果
引入模块
bs.loadScript('../', ['boot-tree'])
bs.loadStyle('../', ['boot-tree']);
此标签只能通过JS方法进行初始化
Usage:

<div id="tree" style="width: 250px;height: 300px;overflow: auto;"></div>

$('#tree').treeview({data: d});
Data Format(数据格式):
[
  {
    text: "Parent 1",
    nodes: [
      {
        text: "Child 1",
        nodes: [
          {
            text: "Grandchild 1"
          },
          {
            text: "Grandchild 2"
          }
        ]
      },
      {
        text: "Child 2"
      }
    ]
  },
  {
    text: "Parent 2"
  },
  {
    text: "Parent 3"
  },
  {
    text: "Parent 4"
  },
  {
    text: "Parent 5"
  }
];
Tree Data Property(节点属性):
Name Type Description Default
text String(必选项) 列表树节点上的文本，通常是节点右边的小图标 空
icon String(可选项) 列表树节点上的图标，通常是节点左边的图标。(参见bootstrap icon) 空
selectedIcon String(可选项) 当某个节点被选择后显示的图标，通常是节点左边的图标 空
href String(可选项) 结合全局enableLinks选项为列表树节点指定URL 空
selectable Boolean 指定列表树的节点是否可选择。设置为false将使节点展开，并且不能被选择 true
state Object(可选项) 一个节点的初始状态。 空
state.checked Boolean 指示一个节点是否处于checked状态，用一个checkbox图标表示 false
state.disabled Boolean 指示一个节点是否处于disabled状态。（不是selectable，expandable或checkable） false
state.expanded Boolean 指示一个节点是否处于展开状态 false
state.selected Boolean 指示一个节点是否可以被选择 false
color String. Optional 节点的前景色，覆盖全局的前景色选项 空
backColor String. Optional 节点的背景色，覆盖全局的背景色选项。 空
tags Array of Strings. Optional 通过结合全局showTags选项来在列表树节点的右边添加额外的信息。 空

Property:
Name Type Description Default
data Array 列表树上显示的数据 []
backColor String 设置列表树容器的边框颜色，如果不想要边框可以设置showBorder属性为false inherits from Bootstrap.css
checkedIcon String：class名称 设置处于checked状态的复选框图标 glyphicon glyphicon-check
collapseIcon String：class名称 设置列表树可收缩节点的图标 glyphicon glyphicon-minus
color String 设置列表树所有节点的前景色 inherits from Bootstrap.css
emptyIcon String：class名称 设置列表树中没有子节点的节点的图标 glyphicon
enableLinks Boolean 是否使用当前节点的文本作为超链接。超链接的href值必须在每个节点的data结构中给出 false
expandIcon String：class名称 设置列表树可展开节点的图标 glyphicon glyphicon-plus
highlightSearchResults Boolean 是否高亮搜索结果 true
highlightSelected Boolean 当选择节点时是否高亮显示 true
onhoverColor String 设置列表树的节点在用户鼠标滑过时的背景颜色 #F5F5F5
levels Integer 设置继承树默认展开的级别 2
multiSelect Boolean 是否可以同时选择多个节点 false
nodeIcon String：class名称 设置所有列表树节点上的默认图标 glyphicon glyphicon-stop
selectedIcon String：class名称 设置所有被选择的节点上的默认图标 glyphicon glyphicon-stop
searchResultBackColor String 设置搜索结果节点的背景颜色 undefined, inherits
searchResultColor String 设置搜索结果节点的前景颜色 #D9534F
selectedBackColor String 设置被选择节点的背景颜色 #428bca
selectedColor String 设置列表树选择节点的背景颜色 #FFFFFF
showBorder Boolean 是否在节点上显示边框 true
showCheckbox Boolean 是否在节点上显示checkboxes false
showIcon Boolean 是否显示节点图标 true
showTags Boolean 是否在每个节点右边显示tags标签。tag值必须在每个列表树的data结构中给出 false
uncheckedIcon String：class名称 设置图标为未选择状态的checkbox图标 glyphicon glyphicon-unchecked
Method:
Name Parameter Description Return Usage
checkAll options 触发nodeChecked事件，传入silent来阻止其它事件 $('#tree').treeview('checkAll')
checkNode node | nodeId, options 选择指定的节点，接收节点或节点ID，触发nodeChecked事件，传入silent来阻止其它事件。 $('#tree').treeview('checkNode', [ nodeId])
clearSearch 清空以前的搜索结果。例如清除它们的高亮状;触发searchCleared事件 $('#tree').treeview('clearSearch')
collapseAll options 折叠所有的节点，折叠整个树;触发nodeCollapsed事件 $('#tree').treeview('collapseAll')
collapseNode (node | nodeId, options) 折叠指定节点和它的子节点。如果不想折叠子节点，可以设置{ ignoreChildren: true };触发nodeCollapsed事件 $('#tree').treeview('collapseNode', [ nodeId, { silent: true, ignoreChildren: false } ])
disableAll options 禁用所有的节点,触发nodeDisabled事件，传入silent来阻止其它事件 $('#tree').treeview('disableAll')
disableNode (node | nodeId, options) 禁用指定的节点，接收节点或节点ID;触发nodeDisabled事件 $('#tree').treeview('disableNode', [ nodeId])
enableAll options 启用所有的节点;触发nodeEnabled事件 $('#tree').treeview('enableAll')
enableNode (node | nodeId, options) 启用指定的节点，接收节点或节点ID;触发nodeEnabled事件 $('#tree').treeview('enableNode', [ nodeId])
expandAll options 展开所有的树节点。也可以展开任何给定级别的树节点；触发nodeExpanded事件 $('#tree').treeview('expandAll', { levels: 2, silent: true })
expandNode (node | nodeId, options) 展开指定的树节点，接收节点或节点ID。也可以展开任何给定级别的树节点；触发nodeExpanded事件 $('#tree').treeview('expandNode', [ nodeId, { levels: 2 } ])
getCollapsed 返回折叠节点的数组 $('#tree').treeview('getCollapsed', nodeId)
getDisabled 返回被禁用节点的数组 $('#tree').treeview('getDisabled', nodeId)
getEnabled 返回可用节点的数组 $('#tree').treeview('getEnabled', nodeId)
getExpanded 返回所有展开节点的数组 $('#tree').treeview('getExpanded', nodeId)
getNode nodeId 返回给定节点ID的单一节点对象 $('#tree').treeview('getNode', nodeId)
getParent node | nodeId 返回给定节点的父节点，如果没有则返回undefined $('#tree').treeview('getParent', node)
getSelected 返回所有被选择节点的数组 $('#tree').treeview('getSelected', nodeId)
getSiblings node | nodeId 返回给定节点的兄弟节点的数组，如果没有则返回undefined $('#tree').treeview('getSiblings', node)
getUnselected 返回没有被选择节点的数组 $('#tree').treeview('getUnselected', nodeId)
remove 移除列表树容器。移除附加的事件、附加对象和额外的html元素 $('#tree').treeview('remove')
revealNode node | nodeId, options 显示一个树节点，展开从这个节点开始到根节点的所有节点；触发nodeExpanded事件 $('#tree').treeview('revealNode', [ nodeId])
search pattern, options 搜索匹配是非常的指定树节点，并高亮它们。返回配平节点的数组；触发searchComplete事件 
$('#tree').treeview('search', [ 'Parent', {
  ignoreCase: true,     // case insensitive
  exactMatch: false,    // like or equals
  revealResults: true,  // reveal matching nodes
}]);
selectNode node | nodeId, options 选择一个给定的树节点，接收节点或节点ID；触发nodeSelected事件 $('#tree').treeview('selectNode', [ nodeId])
toggleNodeChecked node | nodeId, options 切换节点的Check状态;触发nodeChecked事件或nodeUnchecked事件 $('#tree').treeview('toggleNodeChecked', [ nodeId])
toggleNodeDisabled node | nodeId, options 切换一个节点的可用和不可用状态;触发nodeDisabled事件或nodeEnabled事件 $('#tree').treeview('toggleNodeDisabled', [ nodeId])
toggleNodeExpanded node | nodeId, options 切换一个节点的展开和折叠状态;触发nodeExpanded事件或nodeCollapsed事件 $('#tree').treeview('toggleNodeExpanded', [ nodeId])
toggleNodeSelected node | nodeId, options 切换一个节点的选择状态;触发nodeSelected事件或nodeUnselected事件 $('#tree').treeview('toggleNodeSelected', [ nodeId])
uncheckAll Uncheck所有的节点;触发nodeUnchecked事件 $('#tree').treeview('uncheckAll')
uncheckNode node | nodeId, options Uncheck一个给定的节点，接收节点或节点ID;触发nodeUnchecked事件 $('#tree').treeview('uncheckNode', [ nodeId]
unselectNode node | nodeId, options 不选择指定的节点，接收节点或节点ID;触发nodeUnselected事件 $('#tree').treeview('unselectNode', [ nodeId])
Event:
$('#tree').treeview({
  // The naming convention for callback's is to prepend with `on`
  // and capitalize the first letter of the event name
  // e.g. nodeSelected -> onNodeSelected
  onNodeSelected: function(event, data) {
    // 事件代码...
});
or
$('#tree').on('nodeSelected', function(event, data) {
  // 事件代码...
}); 

nodeChecked (event, node)：一个节点被checked。
nodeCollapsed (event, node)：一个节点被折叠。
nodeDisabled (event, node)：一个节点被禁用。
nodeEnabled (event, node)：一个节点被启用。
nodeExpanded (event, node)：一个节点被展开。
nodeSelected (event, node)：一个节点被选择。
nodeUnchecked (event, node)：一个节点被unchecked。
nodeUnselected (event, node)：取消选择一个节点。
searchComplete (event, results)：搜索完成之后触发。
searchCleared (event, results)：搜索结果被清除之后触发。
